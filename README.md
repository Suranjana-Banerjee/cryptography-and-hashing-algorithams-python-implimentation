## 🔐 **Cryptography and Hashing Techniques**

### 📌 **Project Overview**

This project explores cryptography and hashing techniques used to secure data and communication in modern systems. It includes theoretical concepts along with practical implementations of various encryption and hashing algorithms using Python.

The project was developed as part of my Bachelor of Computer Application (BCA) program.

---

### 🎯 **Objectives**

- Understand fundamentals of cryptography and hashing
- Implement encryption algorithms like:
	- RSA
	- DES
- Classical Ciphers (Caesar, Vigenère)
- Explore hashing techniques like:
	- MD5
	- SHA-1
- Analyze security, performance, and real-world applications

---

### 🛠️ **Technologies Used**

- Language: Python 🐍
- IDE: Spyder
- OS: Windows 11
- Libraries:
	- socket
	- datetime
	- Crypto (for DES)

---

### 🔐 **Features**

- **Encryption & Decryption programs**
- **Implementation of classical ciphers:**
	- Caesar Cipher
	- Vigenère Cipher
	- Transposition Cipher
- Modern cryptography:
	- DES (Data Encryption Standard)
	- RSA Algorithm
- Hashing algorithms:
	- MD5
	- SHA-1
- Practical demonstrations of:
	- Data security
	- Secure communication
